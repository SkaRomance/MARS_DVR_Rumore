## 2. SOGLIE NORMATIVE ESATTE

### 2.1 Valori Limite e Valori d'Azione - Art. 183 D.Lgs. 81/2008

**Definizioni normative precise:**

| Grandezza | Valore | Unità | Articolo |
|-----------|--------|-------|----------|
| **Valore limite di esposizione** | LEX,8h = 87 dB(A) | Esposizione giornaliera | Art. 183, c. 1, lett. d) |
| **Valore limite di esposizione (picco)** | LpC,peak = 140 dB(C) | Picco | Art. 183, c. 1, lett. d) |
| **Valore d'azione superiore** | LEX,8h = 85 dB(A) | Esposizione giornaliera | Art. 183, c. 1, lett. c) |
| **Valore d'azione superiore (picco)** | LpC,peak = 137 dB(C) | Picco | Art. 183, c. 1, lett. c) |
| **Valore d'azione inferiore** | LEX,8h = 80 dB(A) | Esposizione giornaliera | Art. 183, c. 1, lett. b) |
| **Valore d'azione inferiore (picco)** | LpC,peak = 135 dB(C) | Picco | Art. 183, c. 1, lett. b) |

### 2.2 Scadenze Temporali

**Art. 183, c. 3 - Deroga temporanea per valore limite:**
> "Se, nonostante l'adozione delle misure di cui all'articolo 185, il valore limite di esposizione è superato, il datore di lavoro elabora e attua un programma di misure per ridurre l'esposizione"

**Obbligo immediato:** Nessuna esposizione deve mai superare il valore limite (87 dB(A)).

**Deroga possibile per valore limite superato:**
- Solo se giustificato da circostanze eccezionali
- Max 6 mesi rinnovabili (Autorizzazione dalla ASL competente)
- Misure integrative obbligatorie
- Informazione lavoratori
- Sorveglianza sanitaria intensificata

### 2.3 Calcolo LEX,8h - Formula ISO 9612

**Formula normativa esatta:**

```
LEX,8h = LAeq,Tm + 10·log10(Tm/T0)

Dove:
- LEX,8h = Livello di esposizione giornaliera personale (dB(A))
- LAeq,Tm = Livello equivalente continuo ponderato A durante Tm (dB(A))
- Tm = Tempo di esposizione effettivo (ore)
- T0 = Tempo di riferimento = 8 ore
```

**Per esposizione a più fasi (comune in contesti lavorativi):**

```
LEX,8h = 10·log10( Σ(Ti/T0)·10^(LAeq,i/10) )

Dove:
- Ti = durata della fase i-esima (ore)
- LAeq,i = livello equivalente della fase i-esima (dB(A))
- Sommatoria su tutte le fasi espositive
```

**Incertezza di misura:**
ISO 9612 richiede la stima dell'incertezza estesa (±dB) con livello di confidenza 95%.
Il valore finale riportato deve includere l'incertezza.

### 3.5 DPI Uditivi - Art. 192

**Requisiti obbligatori Art. 192:**

#### Fascia 1 (≥ 80 dB(A) o ≥ 135 dB(C) peak):
> Art. 192, c. 3: "Il datore di lavoro mette a disposizione idonei DPI uditivi ai lavoratori"

- Dpi disponibili ma NON obbligatori per lavoratore
- Obbligo informativo su uso e necessità
- Possibilità richiesta sorveglianza sanitaria

#### Fascia 2 (≥ 85 dB(A) o ≥ 137 dB(C) peak):
> Art. 192, c. 4: "Il datore di lavoro deve far usare i DPI uditivi ai lavoratori"

- **DPI OBBLIGATORI** (uso non facoltativo)
- Controllo effettivo utilizzazione
- Verifica idoneità DPI scelti

#### Fascia 3 (> 87 dB(A) o > 140 dB(C) peak):
- Dpi obbligatori
- Azioni correttive immediate
- Divieto prosecuzione lavoro senza DPI efficaci

**Requisiti tecnici DPI uditivi (Art. 192, c. 1-2):**

| Requisito | Normativa riferimento |
|-----------|----------------------|
| Marcatura CE | Reg. (UE) 2016/425 |
| Categoria III | D.Lgs. 81/08 Art. 76 |
| Certificato conformità | UNI EN 352-1/2/... |
| Attenuazione dichiarata | UNI EN ISO 4869-1 |
| Metodo MIRE/REAT ISO 4869-2 | Per calcolo attenuazione |
| Prova di idoneità individuale | Art. 192, c. 5 |

**Calcolo attenuazione reale:**

Attenuazione effettiva = Attenuazione dichiarata - Deviazione standard - Margine sicurezza

**Metodi accettati:**
1. **SNR (Single Number Rating):** H-M-L
2. **HML (High-Medium-Low):** Per rumori spettrali
3. **MIRE (Microphone in Real Ear):** Misura soggettiva

**Procedura scelta DPI (Art. 192, c. 5):**

1. Valutazione rumore ambiente (spettro se possibile)
2. Selezione DPI con attenuazione adeguata
3. Verificare: LEX,8h con DPI < 80 dB(A) - obiettivo
4. Prova individuale (comfort, sigillamento)
5. Verifica periodica efficienza
6. Formazione uso e manutenzione

## 6. CASI BORDERLINE E GESTIONE

### 6.1 Caso 1: Esposizione Variabile Settimanale

**Situatione:** Lavoratori con esposizione significativamente diversa giorno per giorno.

**Riferimento normativo:** Art. 184, c. 1, lett. c) e ISO 9612

**Gestione:**

```
Soluzione normativa:

LEX,8h calcolato su base settimanale quando:
- La durata del lavoro settimanale è significativamente differente dalle 40 ore
- L'esposizione varia considerevolmente giorno per giorno

Formula:
LEX,w = 10·log10( Σ(Ti/Tw)·10^(LAeq,i/10) )

Dove Tw = 40 ore (riferimento settimanale)

Soglie settimanali:
- Valore azione inferiore: LEX,w = 80 dB(A)
- Valore azione superiore: LEX,w = 85 dB(A)
- Valore limite: LEX,w = 87 dB(A)

Il sistema DEVE permettere:
1. Inserimento orari giornalieri diversi
2. Calcolo automatico LEX,w
3. Passaggio opzionale LEX,8h vs LEX,w
4. Giustificazione metodo scelto
```

```
1. INTERRAMIENTO IMMEDIATO

2. PROGRAMMA AZIONI

3. MISURE IMMEDIATE

4. SORVEGLIANZA INTENSIFICATA

5. DOCUMENTAZIONE

6. DEROGA (se necessario)

```

### 6.3 Caso 3: Rumore Impulsivo

**Situzione:** Presenza di rumore impulsivo (picchi, impatti, esplosioni)

**Riferimento normativo:** Art. 184, c. 2, lett. m) e ISO 9612

**Gestione:**

```
Definizione normativa:
- Rumore impulsivo: variazioni rapide livello sonoro (> 10 dB in < 1s)
- Rilevamento: picchi > 130 dB(C)
- Parametro: LpC,peak

Misurazione:
- Fonometro classe 1 obbligatorio
- Time weighting "I" (impulse)
- Rilevamento picchi in dB(C)

Soglie picco:
- Valore azione inferiore: 135 dB(C)
- Valore azione superiore: 137 dB(C)
- Valore limite: 140 dB(C)

Obblighi specifici:
- Misura LpC,peak sempre
- Valutazione separata da LEX,8h
- DPI specifici per impulsi
- Formazione specifica

IL SISTEMA DEVE:
- Flag per fasi impulsive
- Campo separato per LpC,peak
- Controllo automatico soglie picco
- Avviso automatico se superamento
- DPI con attenuazione impulsi
```

### 6.4 Caso 4: Rumore a Bassa Frequenza

**Situzione:** Presenza di frequenze basse (< 250 Hz)

**Riferimento normativo:** Art. 184, c. 2, lett. m) (rumori a bassa frequenza)

**Gestione:**

```
Definizione:
- Basse frequenze: < 250 Hz
- Effetti: fastidio, vibrazioni, disturbo sonno

Valutazione:
- Spettro frequenziale (ottave o terzi d'ottava)
- Indici specifici (LCeq - LAeq)
- Se LCeq - LAeq > 10 dB → presenza significativa LF

Metodo di valutazione:
- Non esiste limite numerico specifico in Italia
- Riferimento: DIN 45680, Dutch Guidelines
- Raccomandazione: valutazione qualitativa obbligatoria

Obblighi:
- Menzionare presenza LF in valutazione
- Considerare effetti su salute
- DPI specifici per LF
- Verifica efficacia DPI su spettro

IL SISTEMA DEVE:
- Campo per indicazione LF
- Flag automatico se LCeq - LAeq > 10 dB
- Campo spettro frequenze
- Avviso per valutazione specifica
```

### 6.5 Caso 5: Interazione con Altri Agenti

**Situzione:** Esposizione simultanea a rumore + altri agenti (vibrazioni, sostanze ototossiche)

**Riferimento normativo:** Art. 184, c. 2, lett. f)

**Gestione:**

```
Agenti sinergici:
- Vibrazioni mano-braccio (HAV)
- Vibrazioni corpo intero (WBV)
- Sostanze ototossiche (solventi, farmaci)
- Stress termico (effetto combinato)

Effetto combinato:
- Potenzia danno uditivo
- Abbassa soglia di azione effettiva
- Richiede valutazione combinata

Obblighi:
- Identificare agenti contemporanei
- Documentare interazione
- Considerare protezione combinata
- Informare lavoratori
- Sorveglianza sanitaria specifica

Valutazione:
- Non esistono formule standardizzate
- Approccio precauzionale
- Riduzione esposizione rumore consigliata
- Valutazione qualitativa obbligatoria

IL SISTEMA DEVE:
- Sezione per "altri agenti"
- Flag interazioni
- Riferimento incrociato ad altre valutazioni
- Avviso per valutazione combinata
```

### 6.6 Caso 6: Lavoratori Part-Time

**Situatione:** Lavoratori con tempo parziale

**Riferimento normativo:** ISO 9612

**Gestione:**

```
Principio:
- Esposizione valutata sul tempo effettivo di lavoro
- LEX,8h calcolato normalizzando a 8h
- Non penalizzare part-time

Formula:
LEX,8h = LAeq,Tm + 10·log10(Tm/8)

Dove Tm = tempo esposizione effettivo

Esempio:
- Part-time 4h, LAeq = 85 dB(A)
- LEX,8h = 85 + 10·log10(4/8) = 85 - 3 = 82 dB(A)

Nota importante:
- Se part-time < 4h → calcolare LEX,w (settimanale)
- Giustificare tempo esposizione considerato

IL SISTEMA DEVE:
- Gestire orari personalizzati
- Calcolo automatico normalizzazione
- Opzione per LEX,w
- Documentazione tempo esposizione
```

---

### 6.7 Casolo 7: Telelavoro / Smart Working

**Situatione:** Lavoratori da remoto

**Riferimento normativo:** Art. 182 (esclusioni)

**Gestione:**

```
Esclusione normativa:
- Telelavoro da casa → NON soggetto a valutazione rumore
- Smart working parziale → valutare solo esposizione in sede

Casi particolari:
- Telelavoro con cuffie audio per lungo tempo → valutare rischio
- Call center da casa → valutare esposizione cuffie
- Ufficio home → ambiente domestico

Documentazione:
- Giustificare esclusione dalla valutazione
- Indicare orari smart working
- Se esposizione in sede ridotta → aggiornare calcolo

IL SISTEMA DEVE:
- Flag per smart working
- Calcolo esposizione proporzionale
- Esclusione automatica fasce temporali smart working
- Documentazione motivata
```

## 9. REQUISITI TECNICI DI CALCOLO

### 9.1 Calcolo LEX,8h - Implementazione

**Formula implementativa:**

```
PASSO 1: Calcolo energia per fase

E_i = T_i · 10^(LAeq,i / 10)

Dove:
- T_i = durata fase i in ore
- LAeq,i = livello equivalente fase i in dB(A)

PASSO 2: Somma energie

E_totale = Σ E_i per tutte le fasi

PASSO 3: Normalizzazione 8h

LEX,8h = 10 · log10( E_totale / 8 )

PASSO 4: Calcolo incertezza (ISO 9612)

U = √( u_misura² + u_campionamento² + u_posizione² + u_strategia² )

PASSO 5: Risultato finale

LEX,8h finale = LEX,8h ± U
```

---

### 9.2 Calcolo Effetto DPI

**Formula attenuazione SNR:**

```
PASSO 1: Determinare attenuazione richiesta

Att_req = LEX,8h - 80 + margine_sicurezza

Dove margine_sicurezza ≥ 5 dB (raccomandato)

PASSO 2: Selezionare DPI con SNR adequato

SNR_DPI ≥ Att_req

PASSO 3: Calcolare esposizione con DPI

LEX,8h_DPI = LEX,8h - SNR_DPI + u_DPI

Dove u_DPI è l'incertezza attenuazione (tipicamente ±3 dB)

PASSO 4: Verifica obiettivo

LEX,8h_DPI < 80 dB(A)

Se NO → DPI non adeguato, selezionare altro
```

---

### 9.3 Calcolo Incertezza

**Componenti incertezza (ISO 9612):**

```
u_misura = incertezza strumento + calibrazione
          (tipicamente ±0.7 dB per classe 1, ±1.5 dB per classe 2)

u_posizione = variabilità posizione microfono
             (stimata o calcolata)

u_campionamento = variabilità temporale
                 (calcolata da misure ripetute)

u_strategia = incertezza metodo strategia
             (task-based, job-based, full-day)

U_totale = √( u_misura² + u_posizione² + u_campionamento² + u_strategia² )

U_estesa = k · U_totale (k = 2 per 95% confidenza)
```
